1.install angular/cli
npm install -g @angular/cli

2.create new project SmartSystemAngular
ng new SmartSystemAngular

3. move in folder project SmartSystemAngular
cd SmartSystemAngular

4.Install the dependencies in the local node_modules folder
npm install

5.Run `ng generate component component-name` to generate a new TooltipTrigger
ng generate component TooltipTrigger

6.install bootstrap@4.0.0-beta.2
npm install bootstrap@4.0.0-beta.2

7.install NG Bootstrap Angular directives specific to Bootstrap 4
npm install @ng-bootstrap/ng-bootstrap --save

8. Build the project. 
ng build

9.Development server Navigate to `http://localhost:4200/
ng serve
